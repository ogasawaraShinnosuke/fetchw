# custom-sorts

This library is fetch utils.

## Installation

```shell
npm install fetch2
```

### or

```shell
yarn add fetch2
```

## Usage

This is all you need to get started:

```ts
const fetch2 = require('fetch2');
```

### or es6

```ts
import { api } from 'fetch2';
```
