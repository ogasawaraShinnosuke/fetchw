# custom-sorts

This library is fetch utils.

## Installation

```shell
npm install fetchw
```

### or

```shell
yarn add fetchw
```

## Usage

This is all you need to get started:

```ts
const fetchw = require('fetchw');
fetchw.get('/');
```

### or es6

```ts
import { get } from 'fetchw';
get('/');
```
